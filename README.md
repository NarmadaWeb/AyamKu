# Ayam Segar

A new Flutter project.

## Web Support

This project now supports the Web platform. You can run and build the web version using the provided scripts.

### Running on Web

To run the application on the web using Chrome:

```bash
./scripts/run_web.sh
```

### Building for Web

To build the release version of the web application:

```bash
./scripts/build_web.sh
```

The build artifacts will be located in the `build/web` directory.
